# 《2.6 整式的加减》教学设计

## 一、课题信息

**课题：** 2.6 整式的加减
**所在章节：** 第二章 整式的加减
**课时：** 1课时，约45分钟
**课型：** 新授课
**适用对象：** 七年级上学期（第一章有理数已学完：正负数、数轴、相反数、绝对值、四则运算与乘方全部掌握；第二章已学整式、合并同类项、去括号）

---

## 二、教学内容分析

本节课是第二章的收官课，是在学生已经掌握单项式、多项式、同类项、合并同类项以及去括号法则的基础上，将上述知识整合运用于整式加减运算。

教材的核心思路是：

\[
\text{整式的加减} \longrightarrow \underbrace{\text{先打包（加括号）}}_{\text{正确列式}} \longrightarrow \underbrace{\text{去括号}}_{\text{去掉外壳}} \longrightarrow \underbrace{\text{合并同类项}}_{\text{最终化简}}
\]

其中"求两个多项式的差要先给被减式打括号"是全课核心容易出错的关键：减号在括号外，括号内各项必须全部变号，漏变号是最高频错误。

本节融合两个重要应用场景：
1. **化简求值**：先化简再代入，运算量远小于先代入；
2. **几何应用**：用含字母的整式表示面积，再做减法，体会代数工具的几何价值。

上承 2.5 去括号（本课直接复用），下启 2.7 第二章单元复习。

---

## 三、教学目标

### 1. 知识与技能目标

学生能够：

1. 理解整式加减的实质是合并同类项；
2. 掌握求两个多项式和与差时"先打括号再去括号"的规范列式方法；
3. 熟练执行整式加减的一般步骤：去括号→合并同类项；
4. 运用"先化简再代入"的策略完成化简求值；
5. 能将几何面积问题转化为整式加减模型求解。

### 2. 过程与方法目标

经历以下完整认知链条：

\[
\text{打包列式} \rightarrow \text{去括号} \rightarrow \text{合并同类项} \rightarrow \text{化简求值} \rightarrow \text{几何建模}
\]

在对比"先代入硬算"与"先化简再代入"两条路线的过程中，体会化简的价值，培养化繁为简的数学思维习惯。

### 3. 情感与价值目标

通过例题中"x 项恰好抵消"的惊喜时刻，以及几何应用中面积差的整洁表达式，让学生感受整式运算的简洁美与实用性，增强学习整式代数的成就感与兴趣。

---

## 四、教学重点与难点

### 教学重点

1. 整式加减的一般步骤：先去括号，再合并同类项；
2. 求两个多项式的差时，被减式必须加括号，减号在括号外；
3. 去括号法则：括号前为减号时，括号内各项全部变号。

### 教学难点

1. 含系数的括号去除：\(-2(4x-5y)\) 中 \(-2\) 要乘以括号内每一项；
2. 变号的彻底性：减号括号内多项式每项都要变号，不能漏项；
3. 化简求值的顺序意识：先化简后代入，而非直接代入原式；
4. 几何建模时面积式的正确展开与化简。

---

## 五、学情分析

七年级学生已具备以下基础：
- 有理数的四则运算（正负数乘法，符号规则已熟练）；
- 单项式与多项式的概念、同类项的识别；
- 合并同类项（系数相加减，字母部分不变）；
- 去括号法则（2.5 刚学，记忆新鲜）。

预期存在的问题：
1. 去括号时减号后漏变号（高频错误，已在 2.5 出现，本课仍需专项强调）；
2. 求差时忘记给被减式加括号，导致只有第一项变号；
3. 含系数括号 \(k(ax+b)\) 只乘了第一项，漏乘后续项；
4. 化简求值时习惯先代入，不愿先化简；
5. 几何建模时不知道如何用式子表示面积。

## 七、教学过程

### 环节一：和差要打包——为什么先加括号

**时间：6分钟**

#### 教师活动

在几何画板上并排写出两个多项式：

\[
2x - 3y \qquad 5x + 4y
\]

提问：「我们想求这两个多项式的和，直接写 \(2x-3y+5x+4y\) 行不行？」
引导学生发现：两式简单，直接写还好；但若被减式有多项，减号只管到第一项就会出错。

示范规范写法：把每个多项式先用括号"打包"：

\[
(2x - 3y) + (5x + 4y)
\]

在画板上用高亮矩形框分别圈出两个多项式（tween 描边动画逐步出现）。

#### 学生活动

跟随教师在草稿本上写出打包过程，说出去括号后的展开式：

\[
= 2x - 3y + 5x + 4y
\]

再合并同类项：

\[
x \text{ 项}：2x + 5x = 7x \qquad y \text{ 项}：-3y + 4y = y
\]

结果：\(7x + y\)

教师在画板上逐步用不同颜色标注同类项，动画渐显每步中间结果。

#### 设计意图

以最简单的两项多项式求和开场，让学生复现去括号→合并同类项的完整路径，建立"打包→拆包→合并"的操作直觉，同时铺垫后续求差时打包的必要性。

---

### 环节二：一般步骤——流程图串联两步

**时间：6分钟**

#### 教师活动

在几何画板上用流程图方框呈现整式加减的一般步骤（两框加箭头，tween 逐帧渐显）：

\[
\boxed{\text{第一步：去括号}} \longrightarrow \boxed{\text{第二步：合并同类项}}
\]

分别讲解两条去括号法则：

| 括号前符号 | 去括号效果 |
|---|---|
| \(+(\cdots)\) | 各项符号不变 |
| \(-(\cdots)\) | 各项符号全变 |

在画板旁展示对比示例，减号变号项用红色标注：

\[
+(3x - 2y) = 3x - 2y
\]
\[
-(3x - 2y) = -3x + 2y
\]

#### 学生活动

口答练习：直接合并同类项（无括号情形）

\[
3(2x - y) - 2(x - 3y)
\]

去括号：\(6x - 3y - 2x + 6y\)；合并：\(4x + 3y\)

教师在画板展示完整计算，renderCard 显示结论卡片。

#### 设计意图

用流程图将两步操作可视化，建立"有括号一定先去括号"的步骤意识；对比两条法则，为后续含系数括号的例题打好基础。

---

### 环节三：例题精讲——x 项消失的惊喜

**时间：8分钟**

#### 教师活动

**例题：** 计算 \((8x - 7y) - 2(4x - 5y)\)

画板写出原式，括号用高亮矩形框标出，提醒：「注意减号外面还有系数 2，括号内每项都要乘 \(-2\)。」

**第一步：去括号**

\[
-2(4x - 5y) = -8x + 10y
\]

关键点：\(-2 \times (-5y) = +10y\)，符号变为正（红色高亮 \(-5y\) 变号为 \(+10y\)）。

展开后：

\[
= 8x - 7y - 8x + 10y
\]

**第二步：合并同类项**

\[
x \text{ 项}：8x - 8x = 0 \quad \text{（x 项完全抵消！）}
\]
\[
y \text{ 项}：-7y + 10y = 3y
\]

结果：\(3y\)

画板动画：\(8x\) 与 \(-8x\) 加删除线效果后消失，\(3y\) 放大显示；renderCard 追加卡片"惊喜：x 项消失，结果只剩 \(3y\)"（tada 动画）。

#### 学生活动

在草稿本上完整写出计算过程，特别注意 \(-2 \times (-5y)\) 的符号；找出哪个步骤最容易出错。

#### 设计意图

选择 x 项恰好抵消的例题，制造"结果比预期干净"的惊喜感，加深对去括号变号法则的记忆；同时展示含系数括号的规范展开方式。

---

### 环节四：化简求值——先化简省大力气

**时间：8分钟**

#### 教师活动

**例题：** 化简 \((2x^2 - x + 3) - (x^2 - x + 1)\)，并在 \(x = -2\) 时求值。

画板左右分栏，分别标注"法一：直接代入"和"法二：先化简"。

**法一：直接代入**

将 \(x = -2\) 代入原式：

\[
\bigl(2 \times 4 - (-2) + 3\bigr) - \bigl(4 - (-2) + 1\bigr) = (8 + 2 + 3) - (4 + 2 + 1) = 13 - 7 = 6
\]

过程步骤多，数字计算繁琐（画板逐步追加，renderCard 标注"计算繁琐"）。

**法二：先化简**

去括号：

\[
= 2x^2 - x + 3 - x^2 + x - 1
\]

合并同类项（\(x^2\) 项：\(2x^2 - x^2 = x^2\)；\(x\) 项：\(-x + x = 0\)；常数项：\(3 - 1 = 2\)）：

\[
= x^2 + 2
\]

再代入 \(x = -2\)：

\[
(-2)^2 + 2 = 4 + 2 = 6
\]

两列结果相同（均为 6），但法二仅需一步代入。renderCard 追加结论"先化简再代入，事半功倍！"（headShake 效果）。

#### 学生活动

小组讨论：法一和法二哪步容易出错？化简后为什么 x 项消失了？

口头说出化简求值的正确顺序。

#### 设计意图

通过两种方法的实际计算量对比，让学生亲身感受先化简的优势，形成"化简求值先化简"的思维定势。x 项再次消失，强化对同类项合并规律的理解。

---

### 环节五：几何应用——面积差的整式表达

**时间：8分钟**

#### 教师活动

**情境：** 一块大长方形草坪，长为 \((3a + b)\)，宽为 \(2a\)；其中一块小长方形花坛，长为 \(a\)，宽为 \(b\)。求草坪去掉花坛后的面积（用含 \(a\)、\(b\) 的整式表示）。

画板用 addPolyline 画出大矩形（蓝色边框），内部画小矩形（橙色边框，错开一角），标注各边长（动画：大矩形先出现，小矩形后渐显）。

**第一步：求大长方形面积**

\[
S_{\text{大}} = (3a + b) \times 2a = 6a^2 + 2ab
\]

**第二步：求小长方形面积**

\[
S_{\text{小}} = a \times b = ab
\]

**第三步：求面积差**

\[
S_{\text{大}} - S_{\text{小}} = (6a^2 + 2ab) - ab = 6a^2 + 2ab - ab = 6a^2 + ab
\]

注意：两式相减需给第二个式子打括号列式，再去括号。
画板在矩形旁逐步标注面积表达式，renderCard 追加"面积差 \(= 6a^2 + ab\)"（tada 效果）。

#### 学生活动

尝试独立写出 \(S_{\text{大}} - S_{\text{小}}\) 的列式过程，检查是否给被减式打括号。

回答：当 \(a = 2, b = 1\) 时，面积差是多少？（代入：\(6 \times 4 + 2 = 26\)）

#### 设计意图

将抽象的整式加减落地到几何面积模型，让学生体会代数工具解决实际问题的能力；同时再次练习"求差先打括号"的规范操作。

### 环节六：易错辨析与小结——常见坑和本课框架

**时间：5分钟（含课堂小结衔接）**

画板分两栏展示"错 vs 对"经典对比，错误项用红色标注、正确项用绿色渐显：

\[

\]
\[
\text{对}：-(a - b) = -a + b
\]

**易错二：求差时漏打括号**

\[
\text{错}：(3x^2 + 2x - 1) - x^2 - 3x + 4
\]

\text{对}：(3x^2 + 2x - 1) - (x^2 - 3x + 4) = 2x^2 + 5x - 5
\]

renderCard 追加小结卡片。renderTable 呈现步骤总结表格（行逐步渐显）：

| 步骤 | 操作 | 注意事项 |
|---|---|---|
| 第一步 | 去括号 | 减号括号内各项全变号 |
| 第二步 | 合并同类项 | 系数相加减，字母不变 |
| 化简求值 | 先化简再代入 | 化简后式子最简再代入 |

结尾悬念：renderCard 加 headShake「整式加减已收入囊中——下节单元复习，整章关键考点逐一亮相，敢来挑战吗？」

#### 学生活动

判断两个算式对错并说明理由；完成课堂出口题（见第十一部分）。

#### 设计意图

集中呈现两类最高频错误，通过颜色对比强化正确做法的视觉记忆；小结表格给学生留下可供复习的结构化框架；悬念结尾激发下节课期待。

1. 合并同类项；

## 十、分层作业设计

### 基础题

**第1题：** 计算 \((3a^2b - 2ab^2 + 5) + (ab^2 - 3a^2b - 2)\)

参考答案：

去括号：\(= 3a^2b - 2ab^2 + 5 + ab^2 - 3a^2b - 2\)

合并同类项：\(-ab^2 + 3\)

**第2题：** 从多项式 \((3x^2 + 2x - 1)\) 中减去 \((x^2 - 3x + 4)\)，求结果。

参考答案：

\[
(3x^2 + 2x - 1) - (x^2 - 3x + 4) = 3x^2 + 2x - 1 - x^2 + 3x - 4 = 2x^2 + 5x - 5
\]

**第3题：** 化简 \(a - (a - b) + (a + b)\)。

参考答案：

\[
= a - a + b + a + b = a + 2b
\]

### 提高题

**第4题：** 化简 \((3x - 2y) - (x - 4y)\)，并在 \(x = 1, y = -1\) 时求值。

参考答案：

化简：\(= 3x - 2y - x + 4y = 2x + 2y\)

代入：\(2 \times 1 + 2 \times (-1) = 2 - 2 = 0\)

**第5题：** 已知 \(A = 2m^2 - 3m + 1\)，\(B = m^2 + m - 2\)，求 \(2A - B\)。

参考答案：

\[
2A - B = 2(2m^2 - 3m + 1) - (m^2 + m - 2)
\]
\[
= 4m^2 - 6m + 2 - m^2 - m + 2 = 3m^2 - 7m + 4
\]

### 实践题

**第6题（几何建模）：** 一块大长方形场地，长为 \((2a + 3b)\)，宽为 \(a\)；其中一块小长方形区域，长为 \(b\)，宽为 \(a\)。用含 \(a\)、\(b\) 的整式表示大场地去掉小区域后的面积，并在 \(a = 3, b = 1\) 时求具体面积。

参考答案：

\[
S_{\text{大}} = (2a + 3b) \times a = 2a^2 + 3ab
\]
\[
S_{\text{小}} = b \times a = ab
\]
\[
S_{\text{大}} - S_{\text{小}} = (2a^2 + 3ab) - ab = 2a^2 + 2ab
\]

代入 \(a = 3, b = 1\)：\(2 \times 9 + 2 \times 3 \times 1 = 18 + 6 = 24\)

## 十一、课堂评价设计

### 评价维度表

| 评价维度 | 达成标准 | 评价方式 |
|---|---|---|
| 规范列式 | 求差时被减式打括号，格式正确 | 例题书写观察 |
| 去括号准确性 | 减号后各项全部变号，含系数括号不漏项 | 课堂练习批改 |
| 合并同类项 | 字母及指数不变，系数正确相加减 | 口答抽查 |
| 化简求值策略 | 先化简后代入，不先代入硬算 | 解题步骤观察 |
| 几何建模 | 能列出正确面积表达式并化简 | 实践题 |

### 课堂出口题

计算 \((3x - 2y) - 2(x - 3y)\)，并在 \(x = 2, y = -1\) 时求值。

**标准答案：**

去括号：

\[
= 3x - 2y - 2x + 6y
\]

合并：

\[
= x + 4y
\]

代入 \(x = 2, y = -1\)：

\[
= 2 + 4 \times (-1) = 2 - 4 = -2
\]

**评分标准：**

- 被减式打括号且正确去括号（3分）；
- 合并同类项结果 \(x + 4y\) 正确（3分）；
- 代入求值 \(-2\) 正确（4分）。

## 十二、教学时需要重点强调的四句话

1. **求差不打括号，必出错——被减式永远要先加括号再写减号。**
2. **减号进括号，每项都变号——一项都不能漏，含系数的括号先乘再变号。**
3. **化简求值不走捷径——先把式子化到最简，再代入数值，步骤少错误少。**
4. **整式加减的本质只有一件事——合并同类项，字母和指数一律不动。**
