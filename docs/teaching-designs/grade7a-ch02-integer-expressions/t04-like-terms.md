## 二、教学内容分析

本节课是在学生已掌握有理数四则运算、理解单项式与多项式概念的基础上，进一步研究多项式的化简运算。

教材从生活中"归类计数"的直觉出发，引出字母计算中"同类可合并"的需求。核心链条如下：

\[
\text{超市归类情境} \rightarrow \text{同类项定义（两同两无关）} \rightarrow \text{合并法则（分配律逆用）} \rightarrow \text{先化简再代入}
\]

**同类项定义（人教版原文）：** 在多项式中，所含字母相同，且相同字母的指数也分别相同的项，叫做同类项。所有常数项都是同类项。

**合并法则（人教版原文）：** 把同类项的系数相加，字母和字母的指数保持不变，所得结果作为这些同类项合并后的项。

**合并依据：** 分配律逆用。例如：

\[
3x + 5x = (3 + 5)x = 8x
\]

可配合长方形面积拼接模型直观理解。

**本节种子题（已验算，数值唯一依据）：**

\[
4x^2 + 2x + 7 + 3x - 8x^2 - 2
\]

合并过程：\(4x^2 + (-8x^2) = -4x^2\)，\(2x + 3x = 5x\)，\(7 + (-2) = 5\)，结果为

\[
-4x^2 + 5x + 5
\]

验算（\(x = 3\)）：原式 \(= 36 + 6 + 7 + 9 - 72 - 2 = -16\)；化简式 \(= -36 + 15 + 5 = -16\)。✓

**越位约束：** 本节不讲去括号法则（下节课内容），遇有括号的算式仅作悬念引导。

## 三、教学目标

### 1. 知识与技能目标

学生能够：

1. 理解同类项的定义，掌握"两同两无关"判断方法；
2. 知道所有常数项互为同类项；
3. 能正确识别并合并多项式中的同类项；
4. 理解合并同类项的依据是分配律逆用；
5. 掌握"先化简，再代入"的求值策略。

### 2. 过程与方法目标

经历从情境到抽象、从感性到理性的完整认知过程：

\[
\text{生活归类} \rightarrow \text{字母类比} \rightarrow \text{定义提炼} \rightarrow \text{法则建构} \rightarrow \text{应用迁移}
\]

培养学生的分类思想、符号意识和化简能力，体会"找同类、合系数、抄字母"的程序性操作。

### 3. 情感与价值目标

通过超市归类、面积拼接等真实情境，感受数学知识来源于生活；通过"先化简再代入"的效率对比，体会化简运算的价值；通过易错辨析，养成验算习惯和严谨态度。

## 四、教学重点与难点

### 教学重点

1. 同类项的判断——"两同"（字母种类相同，且各字母指数分别相同）；
2. 合并同类项的法则与书写规范（系数相加，字母指数照抄，降幂排列）；
3. 先化简再代入求多项式的值。

### 教学难点

1. "两无关"的内化：与系数无关（如 \(3x^2y\) 与 \(-5yx^2\) 是同类项）、与字母书写顺序无关；
2. 系数为 \(1\) 或 \(-1\) 时的符号处理（\(-x^2\) 的系数是 \(-1\)，不能忽略）；
3. 指数不能相加的禁忌：\(x^2 + x^2 = 2x^2\)，而非 \(2x^4\)。

## 五、学情分析

**已有基础：** 学生已学完第一章有理数，能熟练进行正负数四则运算；已学 2.1—2.3，理解单项式、多项式、系数、次数等概念。

**认知障碍预判：**

1. 容易把"合并"混同于"相乘"，把 \(x^2 + x^2\) 误写为 \(2x^4\)（指数相加）；
2. 看到字母顺序不同（如 \(yx^2\) 与 \(x^2y\)）就误判为非同类项；
3. 系数为 \(1\) 或 \(-1\) 时漏写符号，导致 \(-x^2 + 3x^2\) 算成 \(3x^2\)；
4. 把常数项与含字母项错误合并（如 \(3 + 5x = 8x\)）；
5. 合并完成后不按降幂排列，书写不规范。

**教学对策：** 正例与反例并排呈现，强化辨析；重点环节要求学生说出判断依据，不只写结果。

## 七、教学过程

### 环节一：超市归类——情境导入

**时间：5分钟**

#### 教师活动

投影超市货架场景：货架上有 3 箱苹果、5 箱苹果、2 筐橙子。提问：

> 哪些货物可以合在一起数？苹果和橙子能合并吗？为什么？

引导学生回答后，用字母替换：设苹果为 \(a\)，橙子为 \(b\)，写出算式：

\[
3a + 5a + 2b
\]

追问：\(3a\) 和 \(5a\) 能合并，依据是什么？

\[
3a + 5a = (3 + 5)a = 8a
\]

再追问：\(8a\) 和 \(2b\) 能合并吗？——不能，因为字母不同。

最后板书课题：**合并同类项**。

**可视化钩子（几何画板）：** 用 `addBar` 画两根苹果柱（高3和高5），动画合并为高8的单根柱；橙子柱高2单独保留。

#### 学生活动

观察货架图，口头回答哪些能合并；用字母写出 \(3a + 5a + 2b\)；跟读分配律逆用过程，感受"同类才能合"的直觉。

#### 设计意图

从生活情境出发降低认知门槛，通过"苹果加苹果、不能加橙子"为"同类项"概念埋下直觉根基，为下一环节的定义提炼做好铺垫。

### 环节二：两同两无关——同类项定义

**时间：8分钟**

#### 教师活动

**第一步：呈现定义。** 板书人教版原文：

> 在多项式中，所含字母相同，且相同字母的指数也分别相同的项，叫做同类项。

用红色标注"字母相同"，蓝色标注"指数分别相同"，归纳口诀：**两同两无关**（字母种类相同 + 各字母指数相同；与系数无关、与字母书写顺序无关）。

**第二步：辨析字母顺序无关。** 画板并排展示 \(3x^2y\) 与 \(-5yx^2\)，用箭头逐项标注：
- \(3x^2y\)：\(x\) 指数为 2，\(y\) 指数为 1；
- \(-5yx^2\)：\(x\) 指数为 2，\(y\) 指数为 1。

两项字母种类相同、各指数相同，**是同类项**（系数 3 和 \(-5\) 不影响判断，书写顺序不影响判断）。

**第三步：常数项互为同类项。** 板书：\(3\) 与 \(-7\) 均为常数项，合并得 \(3 + (-7) = -4\)。对比反例：\(3\) 与 \(3x\)——一个不含字母，一个含 \(x\)，**不是同类项**。

**第四步：关键反例。** 并排显示 \(x\)（指数为1）与 \(x^2\)（指数为2），用不同颜色圈出指数，结论：字母相同但指数不同，**不是同类项**，不能合并。

**可视化钩子（几何画板）：** `addText` 将五对项目并排展示，动画逐组亮色标出"字母"列和"指数"列，最终追加判断表。

**判断汇总表（`renderTable`）：**

| 式子对 | 字母是否同 | 指数是否同 | 结论 |
|---|---|---|---|
| \(3x^2y\) 与 \(-5yx^2\) | ✓ | ✓ | 同类项 |
| \(2ab\) 与 \(3ba\) | ✓ | ✓ | 同类项 |
| \(2x^2\) 与 \(2x\) | ✓ | ✗ | 非同类项 |
| \(3\) 与 \(-7\) | 均无字母 | — | 同类项 |
| \(-2mn^2\) 与 \(3m^2n\) | ✓ | ✗ | 非同类项 |

#### 学生活动

跟随教师逐对判断，口头说出理由；完成小白板即时练习：判断 \(x^2y\) 与 \(yx^2\) 是否为同类项（答：是，字母顺序不影响判断）。

#### 设计意图

"两同两无关"口诀将定义压缩为可操作的判断程序；五对辨析题覆盖所有易错场景，学生不仅知道"结论"，还能说出"为什么"。

### 环节三：合并法则——分配律逆用与面积模型

**时间：8分钟**

#### 教师活动

**第一步：分配律逆用。** 板书推导链：

\[
3x + 5x \xrightarrow{\text{提取公因式}x} (3 + 5)x \xrightarrow{\text{计算}} 8x
\]

分步逐行亮出（动画：随 narration 依次渐显），颜色标注系数 3、5 和字母 \(x\)。明确：**法则依据 = 分配律逆用；系数相加，字母和指数不变**。

**第二步：面积模型动画。** 几何画板（`keepAspect: true`）展示：
- 左矩形：宽 \(x\)，长 3，面积 \(3x\)；
- 右矩形：宽 \(x\)，长 5，面积 \(5x\)；
- 动画（`runTween`）将右矩形左移贴合，合并为宽 \(x\)、长 8 的大矩形，面积 \(8x\)。

旁注：\(3x + 5x = (3+5)x = 8x\)，几何与代数完美呼应。

**第三步：法则口诀与书写规范。** 面板展示：

> 合并同类项三步法：**找—合—抄**（找同类项、合系数、抄字母部分）

展示合并步骤预告（以种子题为例，仅标出同类项，不化简，留下一环节完整演示）；面板卡片（效果 `tada`）：

> **法则：系数相加，字母和指数保持不变。**

化简后按**降幂排列**（次数由高到低，常数项写最后）。

**可视化钩子：** 面积拼接动画为本环节核心钩子，让学生在空间层面感受"同底（同字母）才能拼合"。

#### 学生活动

跟随黑板写出 \(3x + 5x\) 的推导；观察面积动画，口头描述两矩形合并的过程；齐读"找—合—抄"口诀三遍。

#### 设计意图

分配律逆用给出合并的数学本质，面积模型赋予几何直觉，口诀固化操作程序，三个角度共同建构合并法则的认知结构。

### 环节四：例题精讲——种子题完整演示

**时间：8分钟**

#### 教师活动

板书种子题：合并同类项

\[
4x^2 + 2x + 7 + 3x - 8x^2 - 2
\]

**第一步：标色找同类项。** 画板依次用三色高亮三组：
- 红色：\(4x^2\) 与 \(-8x^2\)（\(x^2\) 类）；
- 蓝色：\(2x\) 与 \(3x\)（\(x\) 类）；
- 绿色：\(7\) 与 \(-2\)（常数项类）。

动画按组依次渐显（每步 `return Promise`，`anim` 控制），narration 同步说明。

**第二步：逐组合并。** 面板分三行展示：

\[
4x^2 + (-8x^2) = (4 - 8)x^2 = -4x^2
\]

\[
2x + 3x = (2 + 3)x = 5x
\]

\[
7 + (-2) = 5
\]

强调：系数用有理数加法（注意符号），字母和指数**照抄不变**。

**第三步：写出结果与验算。** 降幂排列，最终结果：

\[
-4x^2 + 5x + 5
\]

代入 \(x = 3\) 验算：

\[
\text{原式} = 4 \times 9 + 2 \times 3 + 7 + 3 \times 3 - 8 \times 9 - 2 = 36 + 6 + 7 + 9 - 72 - 2 = -16
\]

\[
\text{化简式} = -4 \times 9 + 5 \times 3 + 5 = -36 + 15 + 5 = -16 \quad \checkmark
\]

面板卡片（效果 `bounceIn`）显示最终结果。

**可视化钩子（几何画板）：** `addText` 写出完整等式，三组同类项用对应颜色标注，视觉上与面板文字互相印证。

#### 学生活动

在答题卡上同步跟做，完成每一步后对照黑板；对 \(-8x^2\) 的系数是 \(-8\)（负号属于系数）这一点，学生口头复述。

#### 设计意图

种子题是本节核心，三步走（标色—合并—写结果）程序化操作贯穿始终；验算环节培养自检习惯，也用数值说明化简的正确性。

---

### 环节五：练习与求值——先化简再代入

**时间：8分钟**

#### 教师活动

**第一步：基础化简练习。** 出题：

\[
5x - 3y + 2x + y
\]

面板展示规范解题步骤：找 \(x\) 类（\(5x, 2x\)）→ 合得 \(7x\)；找 \(y\) 类（\(-3y, +y\)）→ 合得 \(-2y\)；结果：

\[
7x - 2y
\]

验算（\(x=4, y=2\)）：原式 \(= 20 - 6 + 8 + 2 = 24\)；化简式 \(= 28 - 4 = 24\)。✓

**第二步：双项合并练习。** 出题：化简 \(3a^2 - 2ab + 5a^2 + 3ab - 1\)。

展示合并过程：\(a^2\) 类合并得 \(8a^2\)，\(ab\) 类合并得 \(ab\)，常数项 \(-1\) 保留。结果：

\[
8a^2 + ab - 1
\]

**第三步：先化简再代入。** 承接上一题结果，代入 \(a = 2, b = -1\)：

\[
8 \times 4 + 2 \times (-1) - 1 = 32 - 2 - 1 = 29
\]

对比直接代入原式：\(3 \times 4 - 2 \times 2 \times (-1) + 5 \times 4 + 3 \times 2 \times (-1) - 1 = 12 + 4 + 20 - 6 - 1 = 29\)，步骤明显更多、更易出错。

面板卡片（效果 `headShake`）：**先化简，再代入——既简便又准确。**

**可视化钩子（几何画板）：** `renderTable` 展示两题合并对比表（项、类别、合并系数、结果），列出各步计算清单。

#### 学生活动

独立完成第一步练习后展示，互评；跟做第三步代入计算，感受化简前后运算量的差距。

#### 设计意图

两道练习形成梯度（单字母→双字母+常数项），第三步"先化简再代入"直接对比两种路径的效率，让学生主动选择化简策略。

---

### 环节六：易错与小结——辨析陷阱，埋下悬念

**时间：4分钟（余量归入本环节）**

#### 教师活动

**第一步：陷阱一——指数相加。** 面板对比：

| 错误写法 | 正确写法 |
|---|---|
| \(x^2 + x^2 = 2x^4\)（✗） | \(x^2 + x^2 = 2x^2\)（✓） |

用 \(x = 3\) 验算：\(x^2 + x^2 = 9 + 9 = 18 = 2 \times 9 = 2x^2\)；若误写为 \(2x^4 = 162\)，显然错误。画板红色叉号标注错误过程。强调：**合并同类项只加系数，指数绝对不动！**

**第二步：陷阱二——非同类项不能合并。** 展示错误：

\[
2x + 3x^2 = 5x^3 \quad (\text{✗})
\]

分析：\(2x\) 的 \(x\) 指数为 1，\(3x^2\) 的 \(x\) 指数为 2，指数不同，**不是同类项**，不能合并。正确结果：降幂排列写 \(3x^2 + 2x\)，已是最简形式。

面板 `renderTable` 展示易错汇总：

| 错误写法 | 错误原因 | 正确结果 |
|---|---|---|
| \(x^2 + x^2 = 2x^4\) | 把指数也相加了 | \(2x^2\) |
| \(2x + 3x^2 = 5x^3\) | 非同类项强行合并，指数相加 | \(3x^2 + 2x\) |
| \(-ab + 3ab = 3ab\) | 忽略系数 \(-1\) | \(2ab\) |

**第三步：知识树与悬念。** 面板依次追加卡片（`bounceIn`）：同类项定义 → 两同两无关 → 合并法则 → 先化简再代入。

最后悬念卡片：

> 如果多项式里有括号，比如 \(3x - (2x + 1)\)，括号里的项还能直接合并吗？下节课《去括号》见！

**可视化钩子（几何画板）：** `addText` 并排显示 \(x\)（指数1）与 \(x^2\)（指数2），箭头标出两者指数差异，强化"指数不同不是同类项"。

#### 学生活动

判断两个陷阱题的对错并说明理由；在答题卡上补全知识树填空；口头复述"找—合—抄"三步法。

#### 设计意图

集中突破两大易错点，对比正误答案配数值验证，让"为什么错"一目了然；悬念结尾激发对下节课的期待，形成知识衔接。

## 八、课堂小结

请学生完成以下填空（括号内为答案）：

1. 同类项的判断依据是"两同"：所含（**字母**）相同，且相同字母的（**指数**）也分别相同。
2. 同类项的判断与（**系数**）无关，与字母的（**书写顺序**）无关。
3. 合并同类项的法则：系数（**相加**），字母和指数（**保持不变**）。
4. 合并同类项的数学依据是（**分配律**）的逆用。
5. 求多项式的值时，应先（**化简**），再（**代入**），可减少运算量和出错率。
6. \(x^2 + x^2 = \)（**\(2x^2\)**），而非 \(2x^4\)。

知识结构树

合并同类项
│
├─ 同类项定义

│   └─ 两无关：与系数无关、与字母书写顺序无关
│       └─ 特例：所有常数项互为同类项
│
├─ 合并法则
│   ├─ 找：找出所有同类项（标色标号）

│   └─ 抄：字母和指数照抄不变
│
├─ 数学依据
│   └─ 分配律逆用：3x+5x=(3+5)x=8x
│
├─ 书写规范
│   └─ 化简后按降幂排列
│
└─ 应用策略
    └─ 先化简，再代入（求多项式的值）
```

    也分别相同的项，叫做同类项。

    口诀：两同两无关
    两同：① 字母种类相同
          ② 各字母指数分别相同
    两无关：与系数无关  与书写顺序无关
    特例：所有常数项互为同类项

二、合并法则
    系数相加，字母和指数不变
    依据：分配律逆用
    3x + 5x = (3+5)x = 8x

三、合并步骤（找—合—抄）
    找：标出同类项
    合：系数相加
    抄：字母指数照抄

四、例题
    4x² + 2x + 7 + 3x - 8x² - 2
    = (4-8)x² + (2+3)x + (7-2)
    = -4x² + 5x + 5  ↑降幂排列

五、易错提示
    ✗  x² + x² = 2x⁴（指数不能相加）
    ✓  x² + x² = 2x²
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 十、分层作业设计

### 基础题

1. 判断下列各对是否为同类项，并说明理由：

   （1）\(4x^2y\) 与 \(-3yx^2\)；　（2）\(2x\) 与 \(2x^2\)；　（3）\(5\) 与 \(-8\)。

   **参考答案：**（1）是同类项，字母 \(x\)、\(y\) 均相同，\(x\) 指数均为2，\(y\) 指数均为1，书写顺序不影响判断；（2）不是同类项，字母相同但 \(x\) 的指数不同（1≠2）；（3）是同类项，均为常数项。

2. 合并同类项：

   （1）\(7a - 3a + 2a\)　　（2）\(5m^2 - 2m + 3m^2 + m\)　　（3）\(2xy - 3x + xy + 5x\)

   **参考答案：**（1）\(6a\)；（2）\(8m^2 - m\)；（3）\(3xy + 2x\)。

### 提高题

3. 合并同类项：\(4x^2 + 2x + 7 + 3x - 8x^2 - 2\)（须写出每步合并过程，降幂排列）。

   **参考答案：**

   \[
   = (4-8)x^2 + (2+3)x + (7-2) = -4x^2 + 5x + 5
   \]

4. 先化简，再代入求值：\(3a^2 - 2ab + 5a^2 + 3ab - 1\)，其中 \(a = 2\)，\(b = -1\)。

   **参考答案：** 化简得 \(8a^2 + ab - 1\)；代入得 \(8 \times 4 + 2 \times (-1) - 1 = 32 - 2 - 1 = 29\)。

### 实践题

5. 一块长方形场地由两块小长方形拼成，宽均为 \(x\) 米，长分别为 4 米和 6 米。

   （1）用两种方法表示总面积，写出算式；

   （2）说明两种算法的结果为什么相等（从分配律角度解释）。

   **参考答案：** 方法一：\(4x + 6x\)；方法二：\((4+6)x = 10x\)；两式相等，依据是分配律逆用 \(4x + 6x = (4+6)x = 10x\)。

## 十一、课堂评价设计

### 过程性评价维度

| 评价维度 | 达成标准 | 评价方式 |
|---|---|---|
| 概念理解 | 能用"两同两无关"准确判断同类项 | 口头说明理由 |
| 法则掌握 | 能正确合并同类项（系数相加，指数不变，降幂排列） | 答题卡即时反馈 |
| 易错辨析 | 能指出 \(x^2+x^2=2x^4\) 的错误并纠正 | 黑板展示 |
| 应用能力 | 能先化简再代入求多项式的值 | 提高题作答 |

### 课堂出口题（约2分钟）

合并同类项 \(2a^2b - 3ab + a^2b + 4ab\)，然后代入 \(a = 1, b = -2\) 求值。

**标准答案：**

第一步：化简：

\[
2a^2b + a^2b = 3a^2b, \quad -3ab + 4ab = ab
\]

\[
\text{化简结果} = 3a^2b + ab
\]

第二步：代入 \(a = 1, b = -2\)：

\[
3 \times 1^2 \times (-2) + 1 \times (-2) = -6 + (-2) = -8
\]

**评分标准：** 判断同类项正确（2分）；合并系数正确（2分）；代入计算正确（2分）；满分6分。出口题得4分及以上视为本课目标达成。

## 十二、教学时需要重点强调的四句话

1. **合并同类项：只加系数，字母和指数一律不动——\(x^2 + x^2 = 2x^2\)，绝不是 \(2x^4\)。**
2. **判断同类项看"两同"：字母种类相同，且各字母的指数分别相同——缺一不可；与系数无关，与字母书写顺序无关。**
3. **所有常数项互为同类项；含字母的项与常数项永远不是同类项，不能合并。**
4. **求多项式的值：先化简，再代入——先合并同类项能减少运算量，降低出错率。**
