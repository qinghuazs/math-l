# 《数轴》教学设计

## 一、课题信息

**课题：** 数轴
**课时：** 1课时，约45分钟
**课型：** 概念新授课
**适用对象：** 七年级上学期（前两课已学正负数的意义与有理数的分类）

---

## 二、教学内容分析

本节课是人教版七年级上册第一章《有理数》第3课，承接第1课"正负数"与第2课"有理数及其分类"，是本章的第一个重要工具性概念。

学生已经能用正负数描述温度高低、海拔升降等方向相反的量，但还缺乏一个直观的"定位工具"——当需要同时比较多个有理数的大小、位置时，语言描述显得笨拙。数轴就是解决这个问题的利器。

本节课的核心逻辑是：

\[
\text{温度计（竖向刻度尺）}\xrightarrow{\text{横过来、抽象化}}\text{数轴（一条有方向的刻度线）}
\]

教学设计从两个生活情境出发——温度计和马路上的站牌——让学生感受"用一条线上的点来定位数"的需要，进而抽象出数轴的三要素（原点、正方向、单位长度），再练习描点与读点，最后通过"病轴门诊"巩固规范写法。

本节课的难点不在于计算，而在于：学生需要理解"数"和"点"之间的对应关系，以及负分数如 \(-\dfrac{3}{2}\) 的定位方法。

---

## 三、教学目标

### 1. 知识与技能目标

学生能够：

1. 了解数轴的引入背景，理解数轴是描述有理数位置的工具；
2. 说出数轴的三要素：原点、正方向、单位长度；
3. 按四步规范画法画出一条数轴；
4. 把给定的有理数（整数、小数、简单分数）在数轴上描点；
5. 读出数轴上已知点所对应的数；
6. 知道"正数在原点右边、负数在原点左边、0在原点"这条基本规律。

### 2. 过程与方法目标

通过情境→观察→类比→抽象→应用的学习路径，经历以下过程：

\[
\text{温度计刻度}\rightarrow\text{横向数线}\rightarrow\text{三要素抽象}\rightarrow\text{描点/读点}\rightarrow\text{辨析纠错}
\]

培养学生从实物中提取数学模型的抽象能力，以及规范作图的习惯。

### 3. 情感与价值目标

让学生感受数学工具的简洁美——一条线、一个原点、一个方向、一个长度，就能把无穷多个有理数"请上"同一条直线。感受数学对现实问题的高度概括能力。

---

## 四、教学重点与难点

### 教学重点

1. 数轴的三要素及其含义；
2. 数轴的规范画法（四步）；
3. 任何有理数都对应数轴上唯一一个点（描点与读点）；
4. 正数在原点右边、负数在原点左边、0在原点。

### 教学难点

1. 理解三要素缺一不可——漏掉任意一个都不完整；
2. 负分数的描点：如 \(-\dfrac{3}{2} = -1.5\)，需先确认在 \(-1\) 与 \(-2\) 之间，再找中点；
3. 区分"每个有理数对应数轴上一个点"与"数轴上每个点都表示有理数"（后者七年级暂不深究，教师知道即可）。

---

## 五、学情分析

七年级学生刚刚接触有理数，对正负数的直觉来自生活经验（温度计、楼层、银行存取款），对"数和位置的对应"有朴素感受，但还未形成系统的数形结合意识。

可能存在的问题：

1. 画数轴时漏画箭头，或把箭头画在两端；
2. 没有明确标出原点，把"0"随意放在直线某处；
3. 单位长度不一致，靠感觉画刻度；
4. 认为数轴上只能表示整数，不知道小数、分数也可以描点；
5. 负分数描点方向错误，例如把 \(-\dfrac{3}{2}\) 描到 \(-2\) 与 \(-3\) 之间；
6. 对"1.5 和 −1.5 左右对称"有直觉，但说不出为什么。

因此，教学中要先让学生在温度计上"读数"和"找格"，再把这种技能迁移到数轴。

---

## 六、教学准备

教师准备：

- 温度计实物（或温度计挂图/投影）；
- 课件：温度计旋转放倒动画、三要素逐个点亮动画、描点落点动画、病轴展示图；
- 黑板可画大数轴（建议提前用粉笔淡淡画好刻度辅助线）；
- 彩色粉笔（不同颜色标注不同要素）。

学生准备：

- 直尺；
- 草稿本（本节课要求当堂动手画不少于两条数轴）。

## 七、教学过程

### 环节一：情境导入——温度计与站牌

**时间：5分钟**

#### 教师活动

出示温度计图片（或实物），提问：

> 现在是冬天，气温从 \(-3\,^\circ\text{C}\) 升高到 \(5\,^\circ\text{C}\)，上升了几度？你是怎么在温度计上看出来的？

让学生描述读数过程：找到 \(-3\) 的位置，找到 \(5\) 的位置，数格子。

随后换一个情境：

> 公交站牌是 \(0\) 号基准点，向东第1棵树在 \(+3\) 位置，向西第1根电线杆在 \(-2\) 位置。你能在一条线上把它们都标出来吗？

让几位学生到黑板上尝试——他们会自然画出一条线并标点。

#### 学生活动

在草稿本上根据温度情境或站牌情境，自由画一条"定位线"，把数标上去。

#### 教师追问

1. 温度计是竖的，你把它横过来，两种图有什么共同点？
2. 如果不规定"哪边是正"，还能准确描述位置吗？

引导学生说出：需要一个"基准点"和一个"方向"。

#### 设计意图

用两个生活情境制造"需要一条数线"的认知需要，让数轴的引入顺理成章，避免直接给定义。温度计的"横过来"为下一环节的变形动画做铺垫。

---

### 环节二：从温度计到数轴——横过来、抽象化

**时间：7分钟**

#### 教师活动

播放（或用手势演示）动画：温度计缓慢旋转90°，竖向刻度变成横向刻度；\(-3\) 到 \(5\) 的范围扩展成一条向右无限延伸的直线。

展示对比图：

```
温度计（竖）       数轴（横）
    5                         
    |               ←—|—|—|—|—|—|—→
   ...          -3  -2 -1  0  1  2 ... 5
    0
   ...
   -3
```

然后提问：

> 温度计的最低刻度是固定的，但数轴可以向两端无限延伸。这说明数轴能表示的数比温度计多得多——它能表示所有有理数的位置！

引导学生说出：温度计是"竖着的数轴"，数轴是"横着放、去掉边界、无限延长的温度计"。

#### 学生活动

在草稿本上把自己刚才画的"定位线"改造成一条带箭头的直线，加上刻度，并在上面标出 \(-3\,^\circ\text{C}\) 和 \(5\,^\circ\text{C}\) 对应的点。

#### 设计意图

"温度计旋转"这个动画钩子把抽象的数轴和学生熟悉的实物打通，降低概念门槛。学生亲手"改造"这一步，让抽象过程有参与感。

---

### 环节三：数轴三要素与规范画法

**时间：9分钟**

#### 1. 引出三要素

教师在黑板上画出一条完整数轴，然后逐步"拆掉"各要素，让学生判断缺了什么、会怎样：

**拆掉原点标记：**

> 这条线上，你知道0在哪里吗？不知道——那正负数的位置就没有参照了。

**拆掉箭头（正方向）：**

> 向左是正还是向右是正？说不清——同一个点，不同人理解成不同的数。

**单位长度不一致：**

> 左边每格1厘米，右边每格2厘米——同一个"数字1"，两段距离不一样，点的位置就乱了。

由此引出定义：

**数轴的三要素：**

1. **原点**：数轴上表示0的点，是所有正负数位置的基准；
2. **正方向**：通常规定向右为正方向，用箭头标出；
3. **单位长度**：相邻整数之间的距离，全轴必须一致。

用课件"逐个点亮"三要素，视觉上强化印象。

#### 2. 规范画法四步

教师边说边在黑板上演示：

| 步骤 | 操作 |
|------|------|
| 第一步 | 用直尺画一条水平直线 |
| 第二步 | 在直线上取一点，标记为原点 \(O\)，旁边写"0" |
| 第三步 | 规定向右为正方向，在直线右端画箭头 |
| 第四步 | 以原点为中心，向两侧按相等间距标出刻度，写上整数 |

#### 学生活动

学生在草稿本上按四步画一条数轴（范围 \(-3\) 到 \(3\)），教师巡视，检查三要素是否齐全。

#### 教师小结

> 画数轴，四步缺一不可：直线打底，原点定位，箭头定向，刻度均匀。

#### 设计意图

"拆掉要素"的逆向演示比直接给定义更有冲击力——让学生自己发现"缺了这个会怎样"，而不是被动接受规则。

---

### 环节四：描点——把数请上数轴

**时间：8分钟**

#### 教师活动

出示题目：

> 在数轴上描出以下各数对应的点：\(4,\;-2,\;1.5,\;-1.5,\;0\)。

先做示范：

- \(0\) 就在原点，直接标；
- \(4\) 在原点右边4个单位；
- \(-2\) 在原点左边2个单位；
- \(1.5\) 在1和2的中间（因为 \(1.5 = 1 + 0.5\)，距1半个单位）；
- \(-1.5\)：先确认在 \(-1\) 和 \(-2\) 之间，然后找中点。

教师课件展示"描点落点动画"——每说一个数，一个点从数轴上空落下到对应位置。

最后指着 \(1.5\) 和 \(-1.5\) 两个点：

> 这两个点，一个在原点右边1.5格，一个在原点左边1.5格，隔着原点，左右对称，有意思吧？它们之间有什么秘密，我们下节课再揭晓。

**特别强调 \(-\dfrac{3}{2}\) 的描点方法：**

\[
-\dfrac{3}{2} = -1.5
\]

先把分数化成小数，再确认在 \(-1\) 与 \(-2\) 之间，且离两端都是 \(0.5\) 个单位。

易错警示：有同学会把 \(-\dfrac{3}{2}\) 误描到 \(-2\) 和 \(-3\) 之间——那是把分子当成位置了，要注意！

#### 学生活动

学生在草稿本上独立描出上面五个点，互相检查，有问题举手。

#### 设计意图

"落点动画"增强描点的视觉感；1.5与−1.5的对称性在不点破"相反数"名称的前提下埋下悬念，为下一课蓄势。

---

### 环节五：读点——这个点是谁

**时间：7分钟**

#### 教师活动

出示一条数轴（原点、正方向、单位长度已标好），上面有三个已知点 \(A\)、\(B\)、\(C\)：

- \(A\) 在原点左边3个单位处；
- \(B\) 在原点右边2.5个单位处（即第2格和第3格中间偏左半格）；
- \(C\) 在原点。

让学生说出 \(A\)、\(B\)、\(C\) 各对应什么数：

\[
A \text{ 对应 } -3,\quad B \text{ 对应 } 2.5,\quad C \text{ 对应 } 0.
\]

再出示一个提高题：数轴上有一点 \(D\)，它在 \(-1\) 与 \(-2\) 之间，且到 \(-1\) 和 \(-2\) 的距离相等，\(D\) 对应什么数？

\[
D \text{ 对应 } -1.5 = -\dfrac{3}{2}.
\]

引导学生说出解题思路：

1. 确认 \(D\) 在负半轴，所以是负数；
2. 在 \(-1\) 与 \(-2\) 之间的中点，说明距原点 \(1.5\) 个单位；
3. 所以 \(D = -1.5\)。

#### 学生活动

两人一组：一人在草稿本数轴上随意点一个点，另一人读出对应的数；互换角色练习2轮。

#### 归纳小结

> 读点：从点出发，看它在原点哪一侧、距原点多少个单位，右边是正、左边是负。  
> 描点：从数出发，判断正负确定方向，按单位长度数格子，落点。

#### 设计意图

"两人互练"把"读点"和"描点"统一成一套互逆操作，让学生理解数与点的双向对应关系。

---

### 环节六：病轴门诊——易错辨析

**时间：5分钟**

#### 教师活动

课件依次展示三条"病轴"，让学生当"小医生"找病因、开药方：

**病轴1：** 一条直线，有刻度，有0，但两端都没有箭头。

> **病因：** 没有正方向，不知道哪边是正、哪边是负。  
> **药方：** 在正数方向一端画箭头。

**病轴2：** 有箭头、有刻度，但没有标"0"，看不出原点在哪里。

> **病因：** 缺原点，无法确定各数的绝对位置。  
> **药方：** 找到对应0的那个刻度，标上原点 \(O\) 和"0"。

**病轴3：** 有原点、有箭头，但刻度不均匀：0到1是2厘米，1到2是3厘米，2到3是1.5厘米。

> **病因：** 单位长度不一致，各数的点就没有统一的参照，位置混乱。  
> **药方：** 用直尺重新等分刻度。

最后补充第四种易错：

> 有同学说"数轴上只能表示整数，小数和分数没有对应的点"——这是对的吗？

学生回答：不对。刚才 \(1.5\) 和 \(-1.5\) 都已经描上去了。

#### 设计意图

"病轴门诊"把常见错误集中呈现，游戏化的"找病因、开药方"语境比单纯警示更有趣，也更容易记住。

---

## 八、课堂小结

**时间：4分钟**（利用课后剩余时间）

请同学们完成以下填空（口头或写在草稿本上）：

1. 数轴的三要素是________、________和________。
2. 正数在原点的________边，负数在原点的________边，0在________。
3. 在数轴上描点 \(-\dfrac{3}{2}\) 时，应先把它化成小数________，然后在________和________之间找到中点。
4. 数轴上每一个有理数都对应________个点；反过来，数轴上的点是否都表示有理数，这个问题我们以后再探索。

**参考答案：**

1. 原点、正方向、单位长度；
2. 右；左；原点处（或"0点"）；
3. \(-1.5\)；\(-1\) 和 \(-2\)；
4. 唯一1个。

**知识结构树：**

```text
数轴
│
├─ 来源：温度计横放 → 无限延伸 → 抽象成数轴
│
├─ 三要素（缺一不可）
│   ├─ 原点：表示0的点，是基准
│   ├─ 正方向：箭头所指方向（通常向右）
│   └─ 单位长度：全轴统一，相邻整数间距离
│
├─ 规范画法四步
│   ① 画直线 ② 定原点 ③ 画箭头 ④ 标刻度
│
├─ 数与点的对应
│   ├─ 描点：由数找点（正→右，负→左，0→原点）
│   └─ 读点：由点说数（看方向+数格子）
│
└─ 易错点
    ├─ 漏箭头 / 没标原点 / 刻度不均
    ├─ 以为只能描整数
    └─ 负分数描点方向搞反
```

最后以一句话收尾，留悬念：

> 今天 \(1.5\) 和 \(-1.5\) 这两个点默默站在原点两侧，隔着原点遥遥相望，它们之间究竟有什么秘密？下节课我们来揭晓。

## 十、分层作业设计

### 基础题

**第1题：** 画一条数轴，在数轴上描出下列各数对应的点，并用字母标出：

\[
A: 3,\quad B: -1,\quad C: 0,\quad D: -3,\quad E: 2
\]

**参考答案：**

画出数轴后，各点位置：
- \(A\) 在原点右边3个单位；
- \(B\) 在原点左边1个单位；
- \(C\) 就在原点处；
- \(D\) 在原点左边3个单位；
- \(E\) 在原点右边2个单位。

（注：\(A\) 与 \(D\) 关于原点左右对称，距原点均为3个单位。）

---

**第2题：** 判断下列说法是否正确，错误的说出理由：

（1）数轴就是一条画了刻度的直线。

（2）负数都在原点的左边，所以原点左边的点都表示负数。

（3）在数轴上，\(-5\) 在 \(-3\) 的左边。

**参考答案：**

（1）**错误。** 数轴除了要有刻度，还必须有原点（标0）、正方向（箭头）、单位长度一致，缺一不可。

（2）**正确。** 负数都在原点左边；反过来，原点左边的点表示的数都小于 \(0\)，都是负数。（教师注：原点左边的点是否都表示"有理数"是另一个更深的问题，七年级不展开，本题不涉及。）

（3）**正确。** 在数轴上，\(-5\) 在原点左边5个单位，\(-3\) 在原点左边3个单位，\(-5\) 比 \(-3\) 更靠左，所以 \(-5\) 在 \(-3\) 左边。

---

### 提高题

**第3题：** 在数轴上，点 \(P\) 距原点3个单位，求点 \(P\) 表示的数。

**参考答案：**

\(P\) 距原点3个单位，可以在原点右边3个单位，也可以在原点左边3个单位，所以：

\[
P \text{ 表示 } 3 \text{ 或 } -3.
\]

（本题有两个答案，不要漏掉负数方向。）

---

**第4题：** 数轴上有三个点 \(A\)、\(B\)、\(C\)：
- \(A\) 在原点右边 \(2.5\) 个单位；
- \(B\) 在 \(-1\) 与 \(-2\) 之间的中点；
- \(C\) 在原点左边 \(\dfrac{5}{2}\) 个单位。

（1）分别写出 \(A\)、\(B\)、\(C\) 对应的数；
（2）\(A\) 与 \(C\) 有什么关系？（不用说出名字，描述位置关系即可。）

**参考答案：**

（1）
\[
A = 2.5,\quad B = -1.5,\quad C = -\dfrac{5}{2} = -2.5.
\]

验算：\(B\) 在 \(-1\) 与 \(-2\) 中间，\(B = -1 - 0.5 = -1.5\)，正确。\(C = -\dfrac{5}{2} = -2.5\)，在原点左边2.5个单位，正确。

（2）\(A = 2.5\) 在原点右边2.5个单位，\(C = -2.5\) 在原点左边2.5个单位，两点到原点距离相等，关于原点左右对称。

### 实践题

**第5题：** 观察下面这条"问题数轴"，找出其中所有的错误并改正：

> （描述：一条直线，两端都画了箭头；直线上有刻度，但没有标0；刻度从左到右标的是 1, 2, 3, 4, 5，左边没有负数刻度；相邻刻度间距不一致。）

**参考答案：**

错误共三处：

1. **两端都有箭头，错误。** 数轴只在正方向一端画箭头（通常向右），另一端不画箭头。
2. **没有标原点和0，错误。** 需要在合适位置标出原点 \(O\) 并写"0"，才能确定正负数的位置。
3. **刻度间距不一致，错误。** 单位长度必须全轴统一，需要用直尺重新均匀分格。

（此题为开放情境描述题，实际授课可配图。）

### 课堂出口题

**题目：**

（1）请在下面横线上填空：数轴的三要素是________、________、________。

（2）在你的草稿本上画一条数轴，在数轴上描出 \(-2.5\) 对应的点，并标出字母 \(M\)。

（3）数轴上有一点 \(N\)，它在原点左边 \(\dfrac{3}{2}\) 个单位处，\(N\) 对应的数是________。

---

**标准答案：**

（1）原点、正方向、单位长度（顺序不限，三个都说到才满分）。

（2）数轴画法正确（含原点、箭头、均匀刻度），\(M\) 在原点左边2.5个单位处，即 \(-2\) 与 \(-3\) 的中间。

演算：\(-2.5\) 在 \(-2\) 和 \(-3\) 之间，到 \(-2\) 距离 \(0.5\) 个单位，到 \(-3\) 距离 \(0.5\) 个单位，是中点。

（3）

\[
N = -\dfrac{3}{2} = -1.5.
\]

演算：在原点左边，所以是负数；\(\dfrac{3}{2} = 1.5\) 个单位，所以 \(N = -1.5\)。

## 十二、教学时需要重点强调的四句话

1. **数轴三要素缺一不可——没有原点不知道"从哪数起"，没有方向不知道"往哪数"，刻度不均匀数出来的位置就是错的。**

2. **正数在原点右边，负数在原点左边，0就在原点——记住这条规律，描点读点都有了方向。**

3. **负分数描点，先化成小数，再确认夹在哪两个整数之间——\(-\dfrac{3}{2} = -1.5\)，在 \(-1\) 和 \(-2\) 之间，不是 \(-2\) 和 \(-3\) 之间。**

4. **每一个有理数都能在数轴上找到唯一一个点，但今天先掌握这一点就够了——数轴上的点是不是都表示有理数，这是个更深的问题，以后遇到再说。**

整节课的主线：

> 从生活中的"刻度尺"出发，一步一步抽象出数轴，再用数轴给每个有理数找到它在数轴上的"家"。
