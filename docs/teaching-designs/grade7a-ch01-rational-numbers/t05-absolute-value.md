# 《绝对值与大小比较》教学设计

## 一、课题信息

**课题：** 绝对值与大小比较
**课时：** 1课时，约45分钟
**课型：** 概念新授课
**适用对象：** 七年级上学期（前四课已学正负数、有理数分类、数轴、相反数）

---

## 二、教学内容分析

本节课是在学生已经掌握数轴、正负数、相反数的基础上，引入绝对值概念，并以此为工具系统梳理有理数大小比较的方法。

第四课《相反数》结尾已埋下伏笔："到原点距离相等的两个点表示互为相反数"——学生意识到"到原点的距离"是个有用的量，本课就是把它正式命名、定义清楚。

绝对值的定义有两个层次：
- **几何意义**：数轴上表示数 \(a\) 的点与原点的距离，记作 \(|a|\)；
- **代数规则**（分类讨论）：

\[
|a| = \begin{cases} a, & a \gt 0 \\ 0, & a = 0 \\ -a, & a \lt 0 \end{cases}
\]

代数规则是本课初中阶段第一次正式出现**分类讨论**思想，是后续学习的基础。

大小比较分三种情形依次展开：
1. 有理数在数轴上左小右大；
2. 正数大于0，0大于负数，正数大于一切负数；
3. 两个负数，绝对值大的反而小（"反而"二字是核心）。

---

## 三、教学目标

### 1. 知识与技能目标

学生能够：

1. 理解绝对值的几何意义：数轴上表示数 \(a\) 的点到原点的距离；
2. 掌握绝对值的代数规则（正数取本身，负数取相反数，0取0），会进行简单计算；
3. 知道非负性：\(|a|\ge 0\)，且等号成立当且仅当 \(a=0\)；
4. 能用数轴比较有理数的大小；
5. 能比较两个负数的大小（绝对值大的反而小）；
6. 能将若干有理数从小到大排列。

### 2. 过程与方法目标

通过观察、归纳、分类讨论，引导学生经历：

\[
\text{距离直觉} \rightarrow \text{几何定义} \rightarrow \text{代数规则} \rightarrow \text{大小比较工具}
\]

初步感受**分类讨论**的思想：遇到未知符号的 \(a\)，先按正、负、0 分三类再分别处理。

### 3. 情感与价值目标

让学生感受绝对值概念来自生活中"只关心距离，不关心方向"的实际需要，体会数学从具体到抽象、从特殊到一般的认识过程；对易错点（负数比大小方向反）保持警惕，养成验算习惯。

---

## 四、教学重点与难点

### 教学重点

1. 绝对值的几何意义及代数规则；
2. 两个负数比大小的方法（绝对值大的反而小）；
3. 有理数综合排序。

### 教学难点

1. 代数规则中"负数的绝对值是它的相反数"——\(a \lt 0\) 时 \(|a| = -a\)，\(-a\) 是正数，学生容易误以为"绝对值可以是负数"；
2. 两个负数比大小方向与直觉相反，"绝对值大的数反而更小"；
3. 含相反数、绝对值的混合式（如 \(-|-3|\)）的化简。

---

## 五、学情分析

学生已有基础：

- 会用数轴表示有理数，知道数轴三要素；
- 理解相反数定义，知道互为相反数的两数"到原点距离相等"；
- 有基本的正整数大小比较经验，但负数比大小是全新挑战。

可能存在的问题：

1. 认为 \(|a| = a\) 恒成立（忽略 \(a \lt 0\) 的情形）；
2. 比较 \(-3\) 与 \(-5\) 时，因为 \(3 \lt 5\) 就认为 \(-3 \lt -5\)；
3. 遇到 \(|-3|\) 就认为"去掉绝对值符号等于3"，对 \(-|-3|\) 中外面的负号视而不见；
4. \(|a| = 3\) 只想到 \(a = 3\)，漏掉 \(a = -3\)（双解意识薄弱）。

## 六、教学准备

教师可以提前准备：

- 数轴卡纸（标注原点，可贴磁吸数字卡片）；
- 课件：双向等长线段动画、分类三岔流程图、数轴排序动画、温度计对比图；

- 气温预报截图（如某地某天最低气温 \(-5\)℃、另一天 \(-3\)℃）。

## 七、教学过程

### 环节一：情境导入——只关心"跑多远"，不关心"往哪跑"

**时间：5分钟**

#### 教师活动

在数轴上标出原点，演示：小狗甲从原点向右跑3格，落在 \(+3\)；小狗乙从原点向左跑3格，落在 \(-3\)。

提问：

> 甲和乙出发地相同、速度相同、跑的时间相同——谁跑的路更远？

#### 学生活动

学生直觉回答：一样远，都跑了3格。

#### 教师追问

1. 用正负数描述两只小狗的位置，分别是多少？（\(+3\) 和 \(-3\)）
2. 它们的位置数不同，但"跑的路程"一样，说明路程只跟什么有关？（与原点的距离，和方向无关）
3. 上节课我们说互为相反数的两个数"到原点距离相等"——现在3和\(-3\)到原点的距离都是3。这个"3"，数学上给它起了个专门的名字，是什么？

引出课题：**绝对值与大小比较**。

#### 设计意图

用双向跑路情境激活上一课"到原点距离"的伏笔，让学生感受绝对值来自"只关心距离，不关心方向"的实际需要，为几何定义做好直觉铺垫。

---

### 环节二：概念形成——绝对值：到原点的距离

**时间：8分钟**

#### 1. 给出定义

> 数轴上表示数 \(a\) 的点与原点的距离，叫作数 \(a\) 的**绝对值**，记作 \(|a|\)，读作"a的绝对值"。

在黑板数轴上用彩色粉笔画出 \(|-3|\) 和 \(|3|\) 对应的线段——两段等长，均为3个单位。

\[
|-3| = 3, \quad |3| = 3
\]

#### 2. 即时练习

让学生填写：

\[
|2.5| = \underline{\quad}, \quad |{-\tfrac{3}{4}}| = \underline{\quad}, \quad |0| = \underline{\quad}
\]

答案：\(2.5\)、\(\dfrac{3}{4}\)、\(0\)。

#### 3. 揭示非负性

距离永远不为负，因此：

\[
|a| \ge 0
\]

等号成立当且仅当 \(a = 0\)（原点到自身距离为0）。

#### 4. 呼应上一课

互为相反数的两数绝对值相等。例如 \(1.5\) 与 \(-1.5\) 互为相反数，\(|1.5| = |-1.5| = 1.5\)。反过来，绝对值相等且符号相反，则互为相反数——两个概念形成闭环。

#### 设计意图

从数轴几何图形出发给出定义，先建立直觉再引入符号，避免学生死记 \(|a|\) 是"去掉负号"这种错误记忆。

---

### 环节三：代数规则——分正、负、0 三类归纳

**时间：8分钟**

#### 1. 分类讨论引入

提问：

> 如果我只告诉你一个数 \(a\)，不告诉你它是正是负，你能直接写出 \(|a|\) 的表达式吗？

引导学生按符号分三类讨论——这是初中第一次正式的分类讨论，教师板书三叉流程：

```
         a 是多少？
        /     |     \
   a>0        a=0       a<0
    ↓           ↓         ↓
  |a|=a      |a|=0     |a|=-a
```

#### 2. 归纳规则

\[
|a| = \begin{cases} a, & a \gt 0 \\ 0, & a = 0 \\ -a, & a \lt 0 \end{cases}
\]

重点强调：当 \(a \lt 0\) 时，\(-a \gt 0\)，绝对值依然是正数——"\(-a\) 不是负数，它是 \(a\) 的相反数"。

#### 3. 例题巩固

计算 \(\left|-\dfrac{3}{4}\right|\)：

因为 \(-\dfrac{3}{4} \lt 0\)，所以

\[
\left|-\dfrac{3}{4}\right| = -\left(-\dfrac{3}{4}\right) = \dfrac{3}{4}
\]

#### 4. 辨析易错

判断："\(|a| = a\) 恒成立"——**错误**。

当 \(a = -2\) 时，\(|a| = 2 \ne a = -2\)。正确的说法是：当 \(a \ge 0\) 时才有 \(|a| = a\)。

#### 设计意图

分类讨论是整个初中数学的核心思想，本课是它的首次亮相，务必让学生清楚"为什么要分类、怎么分类、分几类"。三叉流程图比文字描述更直观，利于记忆。

---

### 环节四：大小比较之数轴法——左小右大

**时间：7分钟**

#### 1. 数轴排序规则

> 数轴上，左边的数小于右边的数。

由此得到三条推论：

- 正数大于0；
- 0大于负数；
- 正数大于一切负数。

\[
\cdots \lt -3 \lt -2 \lt -1 \lt 0 \lt 1 \lt 2 \lt 3 \lt \cdots
\]

#### 2. 排序动画（可视化）

课件展示：将 \(2,\ -3,\ 0,\ -1.5,\ \dfrac{1}{2}\) 五张数字卡片依次落到数轴对应位置，落定后从左到右自动站成一排：

\[
-3 \lt -1.5 \lt 0 \lt \dfrac{1}{2} \lt 2
\]

#### 3. 课堂练习

让学生独立将以下五个数从小到大排列（60秒内完成）：

\[
-4, \quad 1.5, \quad 0, \quad -0.5, \quad 3
\]

答案：\(-4 \lt -0.5 \lt 0 \lt 1.5 \lt 3\)。

#### 设计意图

数轴排序可视化让"左小右大"从规则变成图像记忆，五个数落点排队的动画兼顾正数、负数、0、小数，覆盖主要题型。

---

### 环节五：两个负数比大小——绝对值大的反而小

**时间：8分钟**

#### 1. 情境引入

课件出示气温预报：某地周一最低气温 \(-3\)℃，周二最低气温 \(-5\)℃。

提问：

> 哪天更冷？对应的温度数哪个更小？

学生回答：周二更冷，\(-5\)℃ 更小。

#### 2. 数轴验证

在数轴上找到 \(-3\) 和 \(-5\) 的位置，\(-5\) 在 \(-3\) 的左边，因此 \(-5 \lt -3\)，即：

\[
-3 \gt -5
\]

#### 3. 绝对值视角

\[
|-3| = 3, \quad |-5| = 5, \quad 3 \lt 5
\]

两个负数中，绝对值较大（\(5 \gt 3\)）的那个数（\(-5\)）反而更小。

归纳规则：

> **两个负数比大小，绝对值大的反而小。**

#### 4. 例题

比较 \(-\dfrac{3}{4}\) 与 \(-\dfrac{4}{5}\) 的大小：

通分：\(-\dfrac{3}{4} = -\dfrac{15}{20}\)，\(-\dfrac{4}{5} = -\dfrac{16}{20}\)。

\[
\left|-\dfrac{3}{4}\right| = \dfrac{15}{20}, \quad \left|-\dfrac{4}{5}\right| = \dfrac{16}{20}
\]

\(\dfrac{15}{20} \lt \dfrac{16}{20}\)，即 \(\left|-\dfrac{3}{4}\right| \lt \left|-\dfrac{4}{5}\right|\)，

因此两个负数中绝对值较小的 \(-\dfrac{3}{4}\) 反而更大：

\[
-\dfrac{3}{4} \gt -\dfrac{4}{5}
\]

#### 5. 温度计对比图（可视化）

课件并排显示两支温度计，\(-5\) 的液柱比 \(-3\) 低——液柱越低温度越低，直观呈现"绝对值大的负数反而更冷（更小）"。

#### 设计意图

气温情境让"哪天更冷"有真实感；数轴验证给出严格依据；温度计对比图强化视觉直觉。"反而"二字在板书中加粗，课堂反复强调。

---

### 环节六：综合练习与易错辨析

**时间：5分钟**

#### 练习1（含相反数与绝对值的混合化简）

比较 \(-(-2)\) 与 \(-|-3|\) 的大小。

解：

\[
-(-2) = 2, \quad -|-3| = -(3) = -3
\]

\[
2 \gt -3
\]

#### 练习2（判断题，四题）

① "\(|a|\) 一定大于0"——**错误**。\(a = 0\) 时 \(|0| = 0\)，不大于0，只能说 \(|a| \ge 0\)。

② "两个数中，绝对值大的那个数一定大"——**错误**。\(|-5| = 5 \gt |3| = 3\)，但 \(-5 \lt 3\)，负数区间恰恰相反。

③ "若 \(|a| = |b|\)，则 \(a = b\)"——**错误**。如 \(|3| = |-3| = 3\)，但 \(3 \ne -3\)（两数可互为相反数）。

④ "互为相反数的两个数绝对值一定相等"——**正确**。设互为相反数，则 \(b = -a\)，\(|b| = |-a| = |a|\)。

#### 设计意图

混合化简题打通相反数与绝对值两个概念；判断题集中清理本课四个典型易错点，即时暴露问题。

## 八、课堂小结

**时间：约4分钟**（含知识树板书）

请学生完成以下填空：

1. 数轴上表示数 \(a\) 的点与原点的距离，叫作 \(a\) 的________，记作________。
2. 负数 \(a\) 的绝对值等于________（用 \(a\) 的相反数表示）。
3. 绝对值的非负性：\(|a|\)________0，等号成立当且仅当 \(a\)________。
4. 两个负数比大小，绝对值________的那个数反而________。
5. 下一步，认识了数、比完了大小，我们要让有理数做"运算"了——负数怎么做加法？

**答案：**

1. 绝对值；\(|a|\)；
2. \(-a\)（即 \(a\) 的相反数）；
3. \(\ge\)；\(= 0\)；
4. 大；小；
5. （悬念，留待下课）。

**知识结构树：**

```text
绝对值与大小比较
│
├─ 绝对值
│   ├─ 几何意义：数轴上该数的点到原点的距离
│   ├─ 记法：|a|，读"a的绝对值"
│   ├─ 代数规则（分三类）
│   │   ├─ a>0：|a|=a
│   │   ├─ a=0：|a|=0
│   │   └─ a<0：|a|=-a
│   └─ 非负性：|a|≥0
│
└─ 有理数大小比较
    ├─ 数轴法：左边的数小于右边的数
    ├─ 正数 > 0 > 负数
    └─ 两负数：绝对值大的反而小
```

## 十、分层作业设计

### 基础题

**第1题** 计算下列绝对值：

\[
(1)\; |-7| \qquad (2)\; |0.6| \qquad (3)\; \left|-\dfrac{5}{8}\right| \qquad (4)\; -|-4|
\]

**参考答案：**

\((1)\; 7\)；\((2)\; 0.6\)；\((3)\; \dfrac{5}{8}\)；\((4)\; -4\)。

---

**第2题** 比较大小，在空格内填 \(\lt\)、\(\gt\) 或 \(=\)：

\[
(1)\; -8 \;\underline{\quad}\; -6 \qquad (2)\; -\dfrac{1}{3} \;\underline{\quad}\; -\dfrac{1}{4} \qquad (3)\; |-5| \;\underline{\quad}\; |5|
\]

**参考答案：**

\((1)\; -8 \lt -6\)（\(|-8| = 8 \gt |-6| = 6\)，两负数绝对值大的反而小）；

\((2)\; -\dfrac{1}{3} \lt -\dfrac{1}{4}\)（\(\left|-\dfrac{1}{3}\right| = \dfrac{1}{3} \gt \left|-\dfrac{1}{4}\right| = \dfrac{1}{4}\)，绝对值大的负数反而小）；

\((3)\; |-5| = |5| = 5\)，故填 \(=\)。

---

**第3题** 将下列各数从小到大排列：

\[
3, \quad -5, \quad 0, \quad -2, \quad \dfrac{3}{2}
\]

**参考答案：**

\[
-5 \lt -2 \lt 0 \lt \dfrac{3}{2} \lt 3
\]

---

### 提高题

**第4题** 已知 \(|a| = 3\)，求 \(a\) 的所有可能值，并说明理由。

**参考答案：**

\(a = 3\) 或 \(a = -3\)。

理由：绝对值表示到原点的距离，数轴上到原点距离为3的点有两个，分别在原点左侧和右侧，对应 \(-3\) 和 \(3\)。

---

**第5题** 化简并比较大小：

\[
-(-3) \;\text{与}\; -|-4|
\]

**参考答案：**

\[
-(-3) = 3, \quad -|-4| = -(4) = -4
\]

因为 \(3 \gt -4\)，所以 \(-(-3) \gt -|-4|\)。

---

**第6题** 若 \(a \lt 0\)，化简 \(|2a|\)，并指出结果的符号。

**参考答案：**

\(a \lt 0\) 时，\(2a \lt 0\)，所以 \(|2a| = -(2a) = -2a\)。

因为 \(a \lt 0\)，所以 \(-2a \gt 0\)，结果为正数。

---

### 实践题

**第7题** 某城市一周最低气温如下（单位：℃）：

\[
-3,\quad 2,\quad -5,\quad 0,\quad -1,\quad 4,\quad -2
\]

（1）哪天最冷？哪天最暖？
（2）将这7个温度从低到高排列。
（3）温差最大是哪两天？温差是多少？（提示：最高温与最低温之差）

**参考答案：**

（1）最冷是 \(-5\)℃ 那天；最暖是 \(4\)℃ 那天。

（2）从低到高：\(-5 \lt -3 \lt -2 \lt -1 \lt 0 \lt 2 \lt 4\)。

（3）最高温 \(4\)℃，最低温 \(-5\)℃，温差 \(= 4 - (-5) = 4 + 5 = 9\)（℃）。

## 十一、课堂评价设计

### 评价维度表

| 评价维度 | 达成标准 |
|---|---|
| 概念理解 | 能用"到原点的距离"解释绝对值，不混淆几何与代数两种表述 |
| 代数运算 | 能正确应用分类规则计算绝对值，含负数 \(a \lt 0\) 时 \(\|a\| = -a\) |
| 大小比较 | 能比较两个负数大小，正确使用"绝对值大的反而小" |
| 综合应用 | 能化简含相反数与绝对值的混合式，排列多个有理数 |
| 表达规范 | 书写不等号方向正确，不出现"去掉负号"等模糊表述 |

### 课堂出口题

（下课前3分钟，全班独立完成）

**出口题：**

（1）计算：\(-\left|-\dfrac{5}{2}\right|\)

（2）比较大小：\(-\dfrac{2}{3}\) 与 \(-\dfrac{3}{4}\)，在空格内填 \(\lt\)、\(\gt\) 或 \(=\)：\(-\dfrac{2}{3}\;\underline{\quad}\;-\dfrac{3}{4}\)

（3）将 \(-2,\ 1,\ -\dfrac{1}{2},\ 0,\ -3\) 从小到大排列。

**标准答案：**

（1）\(-\left|-\dfrac{5}{2}\right| = -\dfrac{5}{2}\)

（说明：\(\left|-\dfrac{5}{2}\right| = \dfrac{5}{2}\)，再取相反数得 \(-\dfrac{5}{2}\)。）

（2）\(\left|-\dfrac{2}{3}\right| = \dfrac{2}{3} = \dfrac{8}{12}\)，\(\left|-\dfrac{3}{4}\right| = \dfrac{3}{4} = \dfrac{9}{12}\)，前者绝对值较小，故 \(-\dfrac{2}{3} \gt -\dfrac{3}{4}\)。

（3）\(-3 \lt -2 \lt -\dfrac{1}{2} \lt 0 \lt 1\)。

## 十二、教学时需要重点强调的四句话

1. **绝对值是距离，距离永远不为负——\(|a| \ge 0\)，不存在负的绝对值。**
2. **\(a \lt 0\) 时 \(|a| = -a\)：这里的 \(-a\) 是正数，是 \(a\) 的相反数，不是"把负号去掉"。**
3. **两个负数比大小，绝对值大的那个数反而小——"反而"二字必须记住，方向与正数区间相反。**
4. **\(|a| = c\)（\(c \gt 0\)）有两个解：\(a = c\) 或 \(a = -c\)——到原点距离相等的点有两个，不要漏解。**

整节课可以围绕一条主线展开：

> 从"只关心跑多远、不关心往哪跑"出发，给距离起名叫"绝对值"，再用绝对值作工具比较所有有理数的大小——尤其是负数区间那个"反而"。
