# 《正数和负数》教学设计

## 一、课题信息

**课题：** 正数和负数  
**课时：** 1课时，约45分钟  
**课型：** 概念新授课（第一章起始课）  
**适用对象：** 七年级上学期新生（小学阶段只认识非负数，本课是数系的第一次扩充）

---

## 二、教学内容分析

本节课是人教版七年级上册第一章《有理数》的第1课，也是整个初中数学数系扩展的第一步。

小学阶段，学生认识的数只有0和正数（自然数、正分数、正小数）。进入初中，现实生活中大量出现"零下温度""海拔低于海平面""收支亏损"等情况，原来的数不够用了——这正是引入负数的动机。

教材的逻辑线索是：

\[
\text{现实需求（相反意义的量）}\rightarrow \text{引入负数}\rightarrow \text{正数与负数的定义}\rightarrow \text{0 的特殊地位}\rightarrow \text{用正负数表示相反意义的量}
\]

本课选用天气预报、海拔、收支、足球净胜球等贴近学生生活的素材，让学生体会"引入负数是为了表达现实需求"，而非单纯的符号游戏。

核心知识点：

1. 负数的引入动机——表示"具有相反意义的量"；
2. 正数、负数的定义与读写（"+"可省略，"−"绝不可省略）；
3. 0 既不是正数也不是负数；0 是正数与负数的分界，并不只表示"没有"；
4. 能用正数和负数表示实际问题中相反意义的量（先规定哪方为正，另一方即为负）。

---

## 三、教学目标

### 1. 知识与技能目标

学生能够：

1. 了解负数引入的必要性，理解正数、负数的意义；
2. 掌握正数和负数的读写规则（"+"可省、"−"不可省）；
3. 知道 0 既不是正数也不是负数，理解 0 作为基准的含义；
4. 能够在具体情境中先规定正方向，再用正数和负数表示相反意义的量。

### 2. 过程与方法目标

经历从生活实例到数学抽象的过程：

\[
\text{生活情境}\rightarrow \text{发现不够用}\rightarrow \text{引入负数}\rightarrow \text{概念定义}\rightarrow \text{反过来解决问题}
\]

培养学生"用数学描述现实"的建模意识，以及从特殊到一般的归纳能力。

### 3. 情感与价值目标

让学生体会数学知识是在真实需求中生长出来的，感受数学家面对"数不够用"时敢于创造新数的智慧，激发对数学学习的好奇心和探究欲。

---

## 四、教学重点与难点

### 教学重点

1. 理解正数和负数的意义，掌握其读写方法；
2. 知道 0 既不是正数也不是负数；
3. 能在情境中正确用正负数表示相反意义的量。

### 教学难点

1. 理解 0 不只表示"没有"，还可以表示基准（0℃ 是冰水共存的温度，不是"没有温度"）；
2. 表示相反意义的量时，学生容易忘记先规定正方向就直接写符号——要养成"先约定，再表示"的习惯；
3. 区分"不是正数"与"是负数"（漏掉 0 这个特殊情况）。

---

## 五、学情分析

七年级新生刚从小学升入初中，具体情况如下：

**已有基础：**
- 认识自然数、正分数、正小数；
- 了解加减法运算；
- 在生活中已经接触过"零下温度""地下楼层"等语言表达，有隐约的负数前概念。

**潜在困难：**
1. 看到"零下3℃"会说，但写成"−3℃"时不确定"−"号该不该写；
2. 认为"0 就是没有"，难以接受 0℃ 是一个有意义的温度基准；
3. 容易将"不是正数"等同于"是负数"，忽视 0 的独立地位；
4. 表示相反意义的量时，不清楚为什么要先规定正方向。

**教学对策：** 多用直观情境（温度计、海拔条）降低抽象难度，反复追问"0 表示什么意思"，以辨析易错点代替单纯记结论。

---

## 六、教学准备

**教师准备：**

- 天气预报图片或幻灯片（北京某日气温 \(-3℃\) 至 \(3℃\)）；
- 温度计示意图（可动液柱，后续做动态课件）；
- 海拔高度对比条（珠穆朗玛峰 \(8848.86\) 米、吐鲁番盆地 \(-155\) 米）；
- 存折收支记录示例；
- 黑板板书提前布局（见九、板书设计）。

**学生准备：**

- 课前观察并记录一处生活中出现"负数"的场景（气温、楼层、海拔等均可）；
- 带铅笔、橡皮（本课有多处填空练习）。

## 七、教学过程

### 环节一：情境导入——天气预报里的"零下"

**时间：5分钟**

#### 教师活动

出示北京某日天气预报截图，画面显示：

> 今日气温：\(-3℃\) 至 \(3℃\)

板书两组数据：

\[
3℃ \quad -3℃
\]

提问：

> 这两个温度你们在生活中见过吗？"−3℃"你们怎么读？它表示什么意思？

稍作停顿，再问：

> 小学里学过的数里，有"−3"这样的写法吗？

#### 学生活动

学生根据生活经验自由发言：
- "零下三度"；
- "比零低三度"；
- "小学没学过有减号的数"。

#### 教师追问

> 那么，"零上3℃"和"零下3℃"，这两个温度有什么关系？

引导学生说出：**方向相反，数值一样**。

#### 设计意图

用学生熟悉的天气预报切入，让学生感受到负数不是凭空造出来的，而是生活中本来就有这样的需求。同时制造"认知缺口"：小学学的数不够用了。

---

### 环节二：数不够用了——数系扩充小史

**时间：5分钟**

#### 教师活动

继续追问：

> 小学的数（0、1、2、3……以及正分数、正小数）为什么表示不了"零下3℃"？

引导学生理解：原来的数都是"零或比零多"，无法表达"比零少""比基准低""亏损"这类概念。

随后展示三组"相反意义的量"，逐一板书：

| 情境 | 一个方向 | 相反方向 |
|---|---|---|
| 温度 | 零上 3℃ | 零下 3℃ |
| 海拔 | 高于海平面 8848.86 米 | 低于海平面 155 米 |
| 收支 | 收入 500 元 | 支出 200 元 |

提问：

> 能不能用一种简洁的方式，让两个相反方向的数都能写出来？

#### 学生活动

学生尝试表达，教师引导：约定一个方向为"正"，用我们已有的数前加"+"号表示；另一个方向加"−"号表示。

#### 教师揭示

> 这就是数学家的做法。给数前面加"+"或"−"，"+"表示正方向，"−"表示负方向。带"−"号的数，就叫**负数**。

#### 设计意图

帮助学生理解负数不是数学老师强行规定的，而是人类在实际需求中自然创造的工具，渗透"数系扩充源于实践"的数学文化。

---

### 环节三：概念形成——正数与负数

**时间：10分钟**

#### 1. 给出定义

教师板书：

> **正数：** 像 \(1,\ 2.5,\ \dfrac{3}{4},\ 8848.86\) 这样大于 0 的数，叫作**正数**。正数前面可以加"+"号，也可以省略不写。

> **负数：** 在正数前面加上"−"号，就得到**负数**，如 \(-1,\ -2.5,\ -\dfrac{3}{4},\ -155\)。负数前面的"−"号**不可以省略**。

强调两点：

- \(+3\) 和 \(3\) 是同一个数，"+"可以不写；
- \(-3\) 的"−"号必须写，去掉就变成了 \(3\)，意思完全不同。

#### 2. 读法示范

教师示范读法：

\[
+3 \quad \text{读作：正三}
\]

\[
-3 \quad \text{读作：负三}
\]

\[
-155 \quad \text{读作：负一百五十五}
\]

#### 3. 回到情境中写出正负数

逐一板书（已验算数据）：

- 零上3℃记 \(+3℃\)，零下3℃记 \(-3℃\)；
- 珠穆朗玛峰海拔约 \(8848.86\) 米，吐鲁番盆地海拔约 \(-155\) 米；
- 收入500元记 \(+500\) 元，支出200元记 \(-200\) 元；
- 增长6.2%记 \(+6.2\%\)，下降1.5%记 \(-1.5\%\)。

#### 4. 即时练习（口答）

教师依次提问，学生口头回答：

> (1) 足球比赛中，进了3个球，记作？丢了2个球，记作？

答：进3球记 \(+3\)，丢2球记 \(-2\)。

> (2) 电梯上升5层记 \(+5\)，那么下降3层记作？

答：记 \(-3\)。

#### 设计意图

在定义与读写之后立即回到情境，帮助学生把抽象符号和实际意义对应起来，避免学生背了定义但不会用。口答形式节省时间，全班参与。

---

### 环节四：0 的特殊地位

**时间：6分钟**

#### 教师提问

> 0 是正数还是负数？

让学生先独立思考30秒，再发言。

#### 学生活动

学生可能出现的答案：
- "0 不是正数也不是负数"（正确）；
- "0 是正数"（误认为正负分界点属于正方向）；
- "0 就是没有，不算数"（混淆了数量零与数值零）。

#### 教师讲解

**第一层：0 既不是正数也不是负数。**

正数大于 0，负数小于 0，0 本身不满足任何一个条件，所以单独作为一类。

板书：

\[
\text{正数} > 0,\quad \text{负数} < 0,\quad 0 \text{ 既不是正数，也不是负数}
\]

**第二层：0 是正数和负数的分界。**

用温度计直观说明（画简图或展示课件动画）：液柱在 0 刻度以上是正温度，以下是负温度，0 刻度是分界线。

**第三层：0 不只表示"没有"，还可以表示基准。**

> 0℃ 是冰水共存的温度（冰点），不是"没有温度"。
> 海拔 0 米是海平面，不是"没有高度"。
> 存折余额 0 元是账户清零，不是"没有账户"。

追问：

> 如果今天气温是 0℃，你觉得冷不冷？（冷！0℃ 是有意义的温度，不是没有温度。）

#### 设计意图

"0 既不是正数也不是负数"是本课最容易被忽视的知识点，也是后续学习有理数分类的基础。用三个层次递进讲解，配合生活实例，让学生真正理解 0 的特殊地位。

---

### 环节五：基础练习——用正负数表示相反意义的量

**时间：9分钟**

#### 练习1（全班齐做，教师巡视）

用正数或负数填写下表：

| 描述 | 记作 |
|---|---|
| 向东行驶 3 米 | |
| 向西行驶 2 米 | |
| 收入 500 元 | |
| 支出 200 元 | |
| 增长 6.2% | |
| 下降 1.5% | |

**参考答案：**

| 描述 | 记作 |
|---|---|
| 向东行驶 3 米 | \(+3\) 米（或 \(3\) 米） |
| 向西行驶 2 米 | \(-2\) 米 |
| 收入 500 元 | \(+500\) 元（或 \(500\) 元） |
| 支出 200 元 | \(-200\) 元 |
| 增长 6.2% | \(+6.2\%\)（或 \(6.2\%\)） |
| 下降 1.5% | \(-1.5\%\) |

做完后教师追问：

> 在"向东/向西"这组中，我们规定了向东为正——如果换一个人，规定向西为正，那向东 3 米应该记什么？

（答：记 \(-3\) 米。）

引导学生体会：**正负号的选取取决于事先的约定，约定不同，同一件事可以有不同的记法，但两个相反量的符号一定相反。**

#### 练习2（指名板演）

请一位同学上黑板，用正数或负数回答：

> 某运动员体重比标准体重重了 2.5 千克，另一运动员体重比标准体重轻了 1.5 千克。分别用正负数表示这两位运动员体重与标准体重的差。

**参考答案：** 重了 2.5 千克记 \(+2.5\) 千克；轻了 1.5 千克记 \(-1.5\) 千克。

#### 设计意图

表格练习强调"先规定正方向，再记符号"的程序意识。追问换方向的问题，帮助学生理解正负记号的约定性，不是天然固定的。

---

### 环节六：实际应用与易错辨析

**时间：6分钟**

#### 辨析1

> 判断并说明理由：\(-5\) 一定比 \(-2\) 小。

让学生先不借助数轴作判断（数轴是第3课内容，此处暂不提），只从"负数表示相反方向"感性理解：欠债5元比欠债2元"欠得更多"，所以 \(-5\) 比 \(-2\) 小。

说明：此处只要求学生有感性判断，不要求严格证明（大小比较在后续课再展开）。

#### 辨析2

> 以下说法正确吗？
>
> (1) "不是正数的数一定是负数。"
>
> (2) "0 就是什么都没有，不是一个数。"
>
> (3) "负数前面的'−'号可以省略不写。"

学生判断后教师归纳：

- (1) **错误**。0 不是正数，但 0 也不是负数。
- (2) **错误**。0 是一个数，0℃、海拔 0 米都有实际意义；说"没有"是混淆了数量意义和数学中的数值。
- (3) **错误**。负数的"−"号是其本质特征，省略后就成了正数，意思完全改变。

#### 设计意图

三个辨析题对应本课三大易错点，集中在最后6分钟处理，让学生带着"这里容易错"的警觉意识结束新授。

## 八、课堂小结

利用课堂剩余约4分钟，请学生完成以下填空（可口答，也可写在草稿纸上），然后对照答案：

1. 大于 0 的数叫作________，小于 0 的数叫作________，0 既不是________也不是________。
2. 正数前面的"+"号________（填"可以"或"不可以"）省略，负数前面的"−"号________（填"可以"或"不可以"）省略。
3. 0℃ 表示________，不是"没有温度"；海拔 0 米表示________，不是"没有高度"。
4. 用正负数表示相反意义的量，要先________，再分别用正、负数表示两个相反方向的量。

**答案：**

1. 正数；负数；正数；负数；
2. 可以；不可以；
3. 冰水共存的温度（冰点）；海平面；
4. 规定其中一个方向为正方向（正方向）。

**知识结构树：**

```text
正数和负数
│
├─ 引入动机
│   └─ 表示"具有相反意义的量"（温度、海拔、收支……）
│
├─ 正数
│   ├─ 定义：大于 0 的数
│   └─ 写法："+"号可省略
│
├─ 负数
│   ├─ 定义：在正数前加"−"号
│   └─ 写法："−"号绝不可省略
│
├─ 0
│   ├─ 既不是正数，也不是负数
│   ├─ 是正数与负数的分界
│   └─ 可以表示基准（冰点、海平面……），不只是"没有"
│
└─ 应用
    └─ 先规定正方向 → 两个相反量分别用正、负数表示
```

结束时留下悬念：

> 引入负数后，数的家族变大了——0、正数、负数，它们有没有统一的名字，怎么分类整理呢？下节课我们来认识**有理数**。

## 十、分层作业设计

### 基础题

**（适合全体学生，当堂或课后独立完成）**

**第1题** 判断下列说法是否正确，错误的说明原因：

(1) \(-7\) 是负数，\(+7\) 是正数，\(0\) 是正数。  
(2) 负数前面的"−"号可以根据情况省略。  
(3) 零下20℃记作 \(-20℃\)，零上20℃记作 \(+20℃\)（或 \(20℃\)）。

**参考答案：**

(1) **错误**。0 既不是正数，也不是负数。  
(2) **错误**。负数的"−"号是其本质，省略后就变成了正数，不可省略。  
(3) **正确**。

---

**第2题** 用正数或负数填写括号：

(1) 珠穆朗玛峰海拔约 \(8848.86\) 米，记作（　　）米；吐鲁番盆地海拔约低于海平面 \(155\) 米，记作（　　）米。

(2) 小明这个月零花钱收入 \(150\) 元，记作（　　）元；消费 \(80\) 元，记作（　　）元。

**参考答案：**

(1) \(+8848.86\)（或 \(8848.86\)）；\(-155\)。  
(2) \(+150\)（或 \(150\)）；\(-80\)。

---

### 提高题

**（适合学有余力的学生）**

**第3题** 读下面这段话，回答问题：

> 某次数学测验，小华的成绩比班级平均分高 \(8\) 分，小刚的成绩比班级平均分低 \(5\) 分，小丽的成绩恰好等于班级平均分。

(1) 规定高于平均分为正，分别用正数或负数表示三人成绩与平均分的差；  
(2) 若班级平均分是 \(72\) 分，三人的实际成绩各是多少分？

**参考答案：**

(1) 小华：\(+8\) 分；小刚：\(-5\) 分；小丽：\(0\) 分（0 表示与平均分相同）。  
(2) 小华：\(72+8=80\) 分；小刚：\(72+(-5)=67\) 分（即 \(72-5=67\) 分）；小丽：\(72\) 分。

**第4题** 一列火车从 A 站出发，经过若干站到达 B 站。规定从 A 站出发向东为正方向，火车先向东行驶 \(120\) 千米，再向西行驶 \(45\) 千米到达 B 站。

(1) 用正负数表示火车每段路程；  
(2) B 站在 A 站的哪个方向？距 A 站多少千米？

**参考答案：**

(1) 向东 \(120\) 千米记 \(+120\) 千米；向西 \(45\) 千米记 \(-45\) 千米。  
(2) B 站在 A 站以东 \(120-45=75\) 千米处（即正方向 \(75\) 千米）。

## 十一、课堂评价设计

### 评价维度表

| 评价维度 | 达成标准 | 评价方式 |
|---|---|---|
| 概念理解 | 能说出正数、负数的含义，知道 0 的特殊地位 | 口答填空、提问 |
| 符号读写 | 正数"+"可省、负数"−"不可省，读写无误 | 板演、练习1 |
| 情境应用 | 能在具体情境中先规定正方向，再用正负数表示 | 练习1表格、板演 |
| 易错辨析 | 能识别并纠正"0 是正数""−号可省"等典型错误 | 辨析3题 |

### 课堂出口题

下课前2分钟，学生独立完成（写在一张小纸条上交给教师）：

> **出口题：** 某城市某日气温在零下 \(4℃\) 至零上 \(6℃\) 之间。
>
> (1) 用正负数分别表示这两个温度；  
> (2) 这一天气温会经过 \(0℃\) 吗？\(0℃\) 表示什么？它是正数还是负数？

**标准答案：**

(1) 零下 \(4℃\) 记作 \(-4℃\)；零上 \(6℃\) 记作 \(+6℃\)（或 \(6℃\)）。  
(2) 会经过 \(0℃\)（因为气温从 \(-4℃\) 升到 \(6℃\) 必须经过 \(0℃\)）。\(0℃\) 表示冰水共存的温度（冰点），是正数与负数的分界，它**既不是正数，也不是负数**。

## 十二、教学时需要重点强调的四句话

1. **"+"号可省，"−"号不可省——负号是负数的灵魂，去掉它就变成正数了。**
2. **0 既不是正数，也不是负数——它是正数和负数的分界，不能归到任何一边。**
3. **0 不只表示"没有"——0℃ 是冰点，海拔 0 米是海平面，都是有意义的基准。**
4. **表示相反意义的量，先约定正方向，另一个方向自然就是负方向——顺序不能颠倒。**
