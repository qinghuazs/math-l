# 数学动态课件库

面向课堂投影的网页动态课件集合：**无需联网、无需安装，双击即用**。

## 使用方法（老师）

1. 把整个文件夹拷贝到上课电脑；
2. 双击根目录 `index.html`（课程导航页），用 Chrome 或 Edge 打开，点击要上的课；
   也可以直接双击某节课目录下的 `index.html`；
3. 讲课时按翻页笔（或键盘）控制节奏：

| 按键 | 动作 |
|------|------|
| 空格 / → / 翻页笔"下一页" | 下一步 |
| ← / 翻页笔"上一页" | 上一步 |
| 数字 1~7 | 跳到对应教学环节 |
| Home（Mac 笔记本：Fn + ←） | 回到本环节第一步 |
| F | 切换全屏 |

误刷新页面会自动回到刚才的进度。

## 目录结构（开发者）

```
index.html + courses.js         # 课程导航页与课程清单
shared/                         # 唯一一份：课件引擎(core) + 主题(css) + JSXGraph/KaTeX(vendor)
courses/<学段>/<册>/<章>/<节>/   # 每课 = index.html + scenes/ 场景剧本
tests/                          # node --test tests/*.test.js（引擎测试 + 课程结构契约）
docs/                           # 设计文档与实现计划
```

新增一课的流程、场景剧本写法与硬性约定见 `CLAUDE.md`。
