# 指数函数的图像和性质 · 动态演示课件

## 使用方法

1. 把整个文件夹拷贝到上课电脑（无需联网、无需安装任何软件）；
2. 双击 `index.html`，用 Chrome 或 Edge 浏览器打开；
3. 讲课时按翻页笔（或键盘）控制节奏：

| 按键 | 动作 |
|------|------|
| 空格 / → / 翻页笔"下一页" | 下一步 |
| ← / 翻页笔"上一页" | 上一步 |
| 数字 1~7 | 跳到对应教学环节 |
| Home | 回到本环节第一步 |
| F | 切换全屏 |

课件共 7 个环节 32 步，与教案一一对应；顶部圆点可点击跳环节。
误刷新页面会自动回到刚才的位置。
